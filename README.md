🎬 Movie Search Web App (Django + TMDB API)

A simple and modern movie search web application built using **Django** and **TMDB API**, where users can search movies and save their favorite movies.



🚀 Features

- 🔐 User Authentication (Signup / Login / Logout)
- 🔍 Search Movies using TMDB API
- 🎬 View movie details (title, rating, poster, overview)
- ❤️ Add movies to Favorites list
- 👤 Personalized user experience
- ⚡ Fast API-based movie data fetching



🛠️ Tech Stack

- Python
- Django
- HTML5
- CSS3
- TMDB API
- SQLite (default database)
